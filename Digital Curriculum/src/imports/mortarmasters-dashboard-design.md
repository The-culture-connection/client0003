Design a high-engagement learning dashboard for a platform called MORTAR Masters Online. The experience should feel like a video game lobby for entrepreneurs, where users immediately see their next action, achievements, and progress.

The interface must feel motivational, dynamic, and modern, NOT like a corporate analytics dashboard.

The design should prioritize action over information, always guiding users toward their next milestone.

Visual Design Direction

Create a light, modern interface with depth, avoiding flat SaaS dashboard aesthetics.

Use:

• soft shadows
• layered cards
• subtle gradients
• depth and motion cues
• floating widgets

Avoid:

• corporate admin dashboard style
• flat grid layouts
• heavy data tables
• excessive statistics panels

The experience should feel closer to:

• Duolingo
• MasterClass
• Notion
• gaming progression screens

but still professional and refined.

Brand Colors

Primary: #1d1d1d
Accent: #871002
Light Background: off-white / warm neutral
Text: dark charcoal

Accent color should highlight:

• progress
• achievements
• CTA buttons
• active modules

Dashboard Philosophy

The dashboard must always answer:

“What should the user do next?”

Every section should reinforce:

• progress
• achievements
• upcoming actions

Main Dashboard Structure

Design the dashboard like a progression hub or game lobby.

Layout sections:

Hero Action Panel (top)

Learning Journey (modules)

Data Room Asset Progress

Achievements + Next Badge

Upcoming Events

Community Activity

Subtle Leaderboard

Progress Widgets

Each section should appear as modular floating cards.

HERO SECTION — Next Action (Most Important)

Large hero card at top of dashboard.

Content example:

NEXT STEP
Finish Chapter 4 Quiz

Module 3 — Pricing Strategy

Estimated time: 12 minutes

Primary CTA:

Continue Learning

Secondary:

View Module

Design this card to feel rewarding and urgent.

Include:

• progress ring
• lesson icon
• subtle animation indicator

This is a core differentiator of the platform.

Create a checklist-style widget:

DATA ROOM PROGRESS

✔ Business Model Canvas
✔ Customer Interview Guide
◻ Financial Projections
◻ Pitch Deck Draft

Include progress bar showing asset completion percentage.

If assets are missing, show message:

Complete remaining assets to unlock Module Graduation.

This widget should visually reinforce building real business outputs.

Achievements + Next Badge

Badges should feel valuable but not childish.

The UI should feel alive but professional.

• hover elevation on cards
• animated progress rings
• badge glow on hover
• smooth transitions between modules
