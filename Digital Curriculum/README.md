# Digital Curriculum

This is a fresh start for the Digital Curriculum webapp, copied from the UI folder.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

## Authentication UI

The authentication UI flow has been added (no functionality yet):
- Login/Sign Up page at `/login`
- AuthProvider component (mock, no Firebase)
- AuthGuard component (UI only)

All authentication components are in `src/app/components/auth/`.

## Note

This is UI-only. No Firebase or backend functionality has been implemented yet.
