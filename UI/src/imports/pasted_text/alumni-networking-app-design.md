Design a modern, scalable mobile app UI (iOS + Android) for a professional alumni networking platform.

🎨 Design System & Style

Primary color: Black (#000000)

Accent color: Deep Red (#C1121F or similar)

Secondary neutrals: charcoal, off-white, soft gray

Style: sleek, premium, minimal, community-driven (not corporate)

Tone: empowering, social, exclusive but welcoming

Use:

Soft shadows and elevation for cards

Rounded corners (12–20px)

Clean sans-serif font (SF Pro / Inter)

High contrast for accessibility

Components should feel like a mix of:

LinkedIn (professional)

Discord (community/chat)

Tinder (matching/explore)

📱 Required Screens & Flows
1. Alumni Onboarding Flow

Design a multi-step onboarding experience:

Welcome screen (value proposition)

Profile setup:

Name, graduation year, organization

Skills (multi-select chips)

“Skills I can offer” vs “Skills I need”

Networking goals selection:

Mentorship, career, social, collaboration

Optional badges/interests

Progress indicator at top

CTA: “Join the Network”

2. Home / Feed (Groups & Posts)

Feed of posts from groups and users

Post card includes:

Profile picture, name, badge

Text + optional image

Like, comment, share buttons

Floating action button: “Create Post”

Filter tabs:

All / My Groups / Trending

Group preview cards at top

3. Groups Page

List of joined groups + discover groups

Group card includes:

Name, description, member count

Group detail page:

Posts within group

Join/Leave button

Members preview

4. Chat Rooms & Messaging

Chat list screen:

Recent conversations

Unread indicators (red badge)

Chat room UI:

Message bubbles (clean, modern)

Timestamp + read receipts

Input bar with:

Text field

Send button

Optional attachments

Include:

“New Match” chat indicator

5. Events System
Event Creation (Admin/Approved Flow)

Form screen:

Title, date, location, description

Upload flyer

Submit for approval

Approval Workflow (Admin view)

Approve / Reject buttons

Status tags: Pending / Approved / Rejected

Events Feed Page

Event cards:

Image, title, date, location

RSVP button

Filter:

Upcoming / Popular / My Events

6. Explore Page (Core Feature)

This is a high-priority screen.

Swipe or scroll-based discovery

User cards include:

Profile picture

Name, role, organization

Skills offered vs skills needed

Tabs:

Matches

Skill Exchange

Help Wanted

Actions:

Connect

Save

Message

7. Matching Engine UI

“Find Matches” button (prominent CTA)

Results screen:

List of top matches (max 6)

Match reason (e.g., “You have skills they need”)

Match card:

Profile preview

Connect button

8. User Profile

Profile header:

Profile picture

Name, title

Badges (visual icons)

Sections:

About

Skills Offered

Skills Needed

Activity (posts/events)

Editable fields:

All user data

CTA:

Edit Profile

9. Edit Profile Screen

Editable form with:

Profile image upload

Text inputs

Multi-select skill chips

Save button (sticky at bottom)

🧭 Navigation System

Bottom navigation bar with 5 tabs:

Home

Explore

Events

Messages

Profile

Use icons + labels

Highlight active tab in red

🧩 Component System (IMPORTANT)

Create reusable components:

Buttons (primary red, secondary outline)

Cards (post, user, event)

Chips (skills, filters)

Modals (confirmation, approvals)

Badges (achievement indicators)

⚡ Interaction & UX Notes

Smooth transitions and microinteractions

Clear CTAs on every screen

Prioritize:

Networking

Discovery

Engagement

Avoid clutter — keep layouts clean and intentional

🎯 Output Requirement

Generate a complete mobile design system + all screens

Maintain consistency across all screens

Use realistic placeholder data

Ensure designs are developer-ready (clear spacing, hierarchy)
